📊 Proyecto 4: Análisis de Embudo y Retención para MercadoLibre

Este proyecto presenta un análisis de embudo de conversión y retención de usuarios para MercadoLibre, con el objetivo de identificar oportunidades de mejora en adquisición, activación y fidelización de usuarios.

El enfoque del informe sigue la estructura ejecutiva C → F → I (Contexto → Hallazgos → Implicaciones) para facilitar la toma de decisiones estratégicas basadas en datos.

🎯 Objetivo del Proyecto

Analizar el proceso completo del embudo de conversión.

Evaluar métricas de retención por cohortes.

Detectar puntos críticos de abandono.

Proponer acciones concretas para mejorar retención y conversión.

📌 Metodología

El análisis se divide en tres secciones principales:

1️⃣ Contexto

Proceso analizado (embudo completo de usuario).

Periodo de análisis.

Métricas clave evaluadas (activación, conversión, retención).

2️⃣ Hallazgos

Cohortes con mejor desempeño.

3️⃣ Implicaciones

📊 Principales Métricas Analizadas

Retención por cohortes (D1, D7, D30, etc.).

Propuestas concretas para incrementar retención y LTV.
